# TaskFlow

TaskFlow is a React-based task management application that allows users to manage tasks, teams, and boards with role-based access control. Users can create, update, and delete tasks, assign them to team members, and organize them on a board view. The app features authentication, protected routes, and different roles (admin and member) with varying permissions.

## Setup Steps

1. Clone the repository:
   ```
   git clone https://github.com/paskaruchitha72/Paska_Ruchitha_2489647_TaskFlow.git
   cd taskflow
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

## Environment Variables

No environment variables are required for this project. The app uses dummyjson.com for authentication and data.
